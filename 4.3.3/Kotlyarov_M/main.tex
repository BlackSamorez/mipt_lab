\subsection{Определение периода решёток по пространственному спектру}

Проведём измерение параметров дифракционной картины для каждой из шести сеток. Закрепим кассету с сетками вблизи выходного окна лазера, включим излучение и на удалённом экране ($L = 100 \pm 1$~см) наблюдаем дифракционные картины. Для каждой сетки измерим расстояние $l$ между удалёнными максимумами одного направления (горизонтального или вертикального) и число промежутков $m$ между ними. Рассчитаем период каждой решётки по формуле для главных максимумов дифракции на двумерной решётке при малых углах:
\begin{equation}
    d = \frac{(m-1) \lambda L}{l},
    \label{eq:d_spectrum}
\end{equation}
где $m $ --- количество максимумов в отрезке длиной $l$ (по обе стороны от центрального максимума), $\lambda = 532$~нм --- длина волны лазера, $L = 100\pm1$~см --- расстояние от сетки до экрана. Полученные значения $d$ также занесём в таблицу~\ref{tab:periods_with_errors}. Эти значения будем считать эталонными при последующем сравнении методов.

\subsection{Определение периода по увеличенному изображению (модель микроскопа)}

Соберём модель проекционного микроскопа. Отцентрируем систему так, чтобы пучок после прохождения линз попадал в ту же точку экрана. Расположим длиннофокусную линзу Л$_1$ ($F_1 = 110$~мм) на расстоянии $a_1$, чуть большем фокусного, от сетки; с помощью линзы Л$_2$ получим резкое увеличенное изображение сетки на экране. Для исключения влияния саморепродукции сфокусируемся на изображении тонкой проволочки, нанесённой на одну из сеток.

Измерим геометрические параметры системы: $a_1$, $b_1$ (расстояния от сетки до Л$_1$ и от Л$_1$ до промежуточного изображения), $a_2$, $b_2$ (от Л$_2$ до промежуточного изображения и до экрана). Измерим также суммарное расстояние $b_1 + a_2$ (``длина тубуса''). Из формулы тонкой линзы $a_{2}=\frac{b_{2} F_{2}}{b_{2}-F_{2}}$. Увеличение микроскопа $\Gamma=\frac{b_{1} b_{2}}{a_{1} a_{2}}$. Для каждой сетки измерим период увеличенного изображения на экране $d_{\text{экрана}}$ (расстояние между соседними линиями). Рассчитаем действительный период сетки:
\begin{equation}
    d_{\text{микро}} = \frac{d_{\text{экрана}}}{\Gamma}.
    \label{eq:d_micro}
\end{equation}

\subsection{Определение периода по разрешающей способности (метод Аббе)}

Поместим щелевую диафрагму с микрометрическим винтом в заднюю фокальную плоскость~$F$ линзы~$Л_1$ (объектива модели микроскопа). Для каждой сетки будем уменьшать ширину щели~$D$, наблюдая за вторичным изображением на экране. Зафиксируем минимальное значение~$D$, при котором ещё различимо периодическое изображение сетки, то есть щель пропускает дифракционные максимумы порядков $m = 0, \pm 1$ (при дальнейшем сужении щели изображение вырождается в одномерную структуру).

Для третей сетки измерили минимальную ширину щели $D = 2{,}17$~мм. Рассчитаем период решётки по формуле Аббе для когерентного освещения:

\subsection{Пространственная фильтрация и мультиплицирование (качественные наблюдения)}

Проведём опыт по пространственной фильтрации. Подберём сетку со средним периодом и установим щель в фокальной плоскости $F$ так, чтобы она пропускала максимум нулевого порядка, но блокировала максимумы первого порядка в поперечном направлении. Поворачивая щель, получим изображения при трёх ориентациях:
\begin{itemize}
    \item \textit{Вертикальная щель}: пропускает максимумы $(0, m_y)$ --- на экране наблюдается решётка с горизонтальными штрихами;
    \item \textit{Горизонтальная щель}: пропускает максимумы $(m_x, 0)$ --- изображение с вертикальными штрихами;
    \item \textit{Щель под углом $45^\circ$}: пропускает максимумы с $m_x = m_y$ --- на экране возникает наклонная периодическая структура, отсутствующая в исходном объекте.
\end{itemize}
Наблюдаемые явления объясняются тем, что щель в фурье-плоскости осуществляет фильтрацию пространственных частот: пропускаются только те гармоники, фурье-образ которых попадает в апертуру щели.

Для наблюдения мультиплицирования поменяем местами сетку $C$ и щель $D$ (рис.~3). Сначала получим резкое изображение щели на экране, затем поместим кассету с сетками в плоскость $F$. При правильной подборке ширины входной щели $D$ и периода сетки на экране наблюдается многократное повторение изображения щели --- мультиплицирование. Уменьшение ширины щели приводит к расширению её фурье-образа, что облегчает ``рассечение'' спектра сеткой и увеличивает число повторений изображения.

\begin{itemize}
    \item Для сетки №\,1 периоды, определённые по спектру ($d_{\text{спектр}} = 9{,}875 \pm 0{,}101$~мкм) и по увеличенному изображению ($d_{\text{микро}} = 10{,}95 \pm 0{,}016$~мкм), расходятся на $\delta \approx 10{,}9\,\%$.

    Это расхождение превышает оценённые случайные погрешности каждого метода, что указывает на наличие систематических ошибок. Наиболее вероятные причины: неточная центровка линз при сборке модели микроскопа, погрешность определения положения промежуточного изображения, а также возможное отклонение от условия малых углов при расчёте по формуле~\eqref{eq:d_spectrum}.
    
    \item Для сетки №\,2 расхождение между спектральным методом и методом микроскопа составляет $\sim 15\,\%$, что также превышает оценённые случайные погрешности. Вероятная причина --- систематическая ошибка при измерении геометрических параметров $a_1$, $b_1$, $a_2$, $b_2$ (неточная центровка линз, погрешность определения положения промежуточного изображения).
    
    \item Для сетки №\,3 все три метода дают значения периода, согласующиеся в пределах $5$--$7\,\%$. Наибольшее расхождение наблюдается между методом Аббе и спектральным методом, что может быть связано с субъективностью определения ``минимального видимого изображения'' при измерении ширины щели~$D$.
\end{itemize}

\subsection{Анализ погрешностей и источников неопределённости}

Основные вклады в суммарную погрешность измерений:
\begin{enumerate}
    \item \textbf{Инструментальная погрешность линейки} ($\sigma_{\text{сист}} = 0{,}5$~мм) --- доминирующий источник для всех методов, основанных на линейных измерениях. Для спектрального метода это приводит к относительной погрешности $\varepsilon_d \approx 1\,\%$ (табл.~\ref{tab:periods_with_errors}).
    
    \item \textbf{Субъективный критерий ``видимости''} в методе Аббе --- определение минимальной ширины щели $D$, при которой ``ещё видно изображение'', зависит от наблюдателя и условий освещения. Это вносит неучтённую систематическую погрешность, которая может достигать $5$--$10\,\%$.
    
    \item \textbf{Неточность установки щели в фокальной плоскости} $F$ --- если щель смещена относительно задней фокальной плоскости линзы Л$_1$, формула~\eqref{eq:d_abbe_final} перестаёт быть строго справедливой, так как изменяется эффективный апертурный угол.
    
    \item \textbf{Приближение малых углов} в формуле~\eqref{eq:d_spectrum} --- для сеток с малым периодом и больших порядков $m$ угол $\theta$ может достигать $10^\circ$--$15^\circ$, при которых $\sin\theta \approx \tan\theta$ даёт погрешность до $2$--$3\,\%$.
    
    \item \textbf{Погрешность фокусных расстояний} линз --- значения $F_1 = 110$~мм и $F_2 = 25$~мм указаны с точностью $\sim 5\,\%$, что напрямую влияет на расчёт увеличения $\Gamma$ и периода по методу Аббе.
\end{enumerate}

\section{Выводы}

\begin{enumerate}
    \item \textbf{По пространственному спектру}: измерены расстояния между дифракционными максимумами для шести сеток; периоды решёток рассчитаны по формуле Фраунгофера с относительной погрешностью $\sim 1\,\%$ (табл.~\ref{tab:periods_with_errors}).
    
    \item \textbf{По увеличенному изображению}: собрана модель проекционного микроскопа, определено увеличение $\Gamma = 6{,}89 \pm 0{,}07$; периоды трёх сеток рассчитаны через геометрическое увеличение, погрешность составила $1{,}5$--$2{,}2\,\%$ (табл.~\ref{tab:periods_microscope}).
    
    \item \textbf{По методу Аббе}: для сетки №\,3 измерена минимальная ширина щели $D = 2{,}17$~мм, период рассчитан по формуле $d = 2F_1\lambda/D$ с погрешностью $0{,}46\,\%$; для остальных сеток измерения ограничены максимальным ходом микрометрического винта.
    
    \item \textbf{Сравнение методов}: значения периода, полученные тремя способами, согласуются в пределах $5$--$15\,\%$; основные расхождения объясняются систематическими погрешностями геометрических измерений и субъективностью визуальных критериев.
    
    \item \textbf{Качественные явления}: продемонстрированы пространственная фильтрация (выделение ориентированных структур через селекцию фурье-компонент) и мультиплицирование (повторение изображения при дискретизации спектра), подтверждающие корпускулярно-волновой дуализм формирования оптических изображений.
\end{enumerate}

Таким образом, в работе экспериментально подтверждена дифракционная природа предела разрешения микроскопа и справедливость формулы Аббе для когерентно освещённых периодических объектов в пределах оценённой экспериментальной погрешности.
